# **Mesto API**

## О проекте:
API для проекта Mesto

## Стэк технологий:
NodeJS, OOP, Express

## Инструкции по запуску:
- Скачать или склонировать репозитори
- Установить зависимости при помощи npm - `npm i`
- Запустить в development режиме - `npm run dev`
- Запустить - `npm run start`
